# 极客博客网站项目大纲

## 项目结构
```
├── index.html              # 主页面 - 博客首页
├── articles.html           # 文章详情页面
├── about.html              # 个人简介页面
├── main.js                 # 全局交互逻辑与工具函数
├── resources/              # 静态资源目录
│   ├── hero-bg-code.jpg         # 首页代码背景图
│   ├── avatar-tech.jpg         # 博主技术头像
│   ├── post-sample-cover.png    # 文章封面示例图
├── interaction.md          # 交互设计规范
├── design.md               # 视觉风格指南
└── outline.md              # 当前项目大纲文档
```

## 页面功能规划

### 首页 (index.html)
**布局结构**：
- 固定导航栏
- 全屏Hero背景区
- 最新文章卡片流
- 技术栈图标墙
- 页脚联系信息

**核心功能**：
- 动态加载最新6篇文章摘要
- 背景粒子动画效果
- 技术栈图标hover旋转特效
- 移动端折叠式导航

### 文章页 (articles.html)
**布局结构**：
- 同站导航栏
- 文章标题区
- Markdown内容渲染区
- 代码高亮区块
- 文章目录侧边栏

**核心功能**：
- URL参数解析文章ID
- 本地Markdown内容渲染
- 动态生成文章目录导航
- 代码片段语法高亮
- 阅读进度指示器

### 关于页 (about.html)
**布局结构**：
- 立体化头像展示区
- 时间轴式经历模块
- 技能雷达图
- 社交联系方式面板

**核心功能**：
- 3D头像旋转交互
- 时间轴动画触发
- 技能数据可视化
- 联系表单基础验证

## 技术实现

### 前端技术栈
- **核心框架**：原生JavaScript(ES6)
- **样式方案**：Tailwind CSS + 内联<style>标签
- **动画引擎**：Anime.js 实现微交互
- **代码高亮**：Highlight.js 集成
- **可视化**：ECharts 精简版

### 数据管理
- 文章数据存储在本地`articles.json`
- 使用`localStorage`缓存已读文章状态
- 技能数据硬编码在about页面脚本中

## 响应式策略
- 移动优先断点设计：<768px / ≥768px / ≥1024px
- 文章目录侧边栏在移动端转换为底部浮动导航
- 技术栈图标墙自适应列数（2/3/4列）
- 图片资源按设备分辨率加载不同尺寸

## 开发优先级

### 第一阶段 - 核心架构
1. 完成三页面基础框架搭建
2. 实现文章Markdown渲染管线
3. 构建响应式导航系统
4. 部署本地数据存储方案

### 第二阶段 - 体验增强
1. 全站交互动效实现
2. 阅读进度跟踪功能
3. 代码高亮深度定制
4. 文章目录自动生成

### 第三阶段 - 优化迭代
1. 首屏加载性能优化
2. 服务端渲染降级方案
3. 离线访问支持
4. 动画性能调优
