# 极客博客网站设计风格

## 设计理念

### 视觉语言
- **代码编辑器美学**: 采用类似VS Code、Sublime Text等现代代码编辑器的深色主题设计
- **开发者友好**: 界面元素和交互模式符合开发者的使用习惯
- **极简主义**: 去除不必要的装饰，专注于内容展示和功能实现
- **专业感**: 通过精细的排版、合适的间距和高质量的设计元素营造专业氛围

### 颜色方案
- **主色调**: 深色背景 (#1e1e1e, #2d2d30)
- **强调色**: 
  - 蓝色 (#007acc) - 主要操作和链接
  - 绿色 (#4ec9b0) - 成功状态和活跃标签
  - 橙色 (#ff8c00) - 警告和重要提示
  - 紫色 (#c586c0) - 特殊标记和高亮
- **文本颜色**: 
  - 主要文本 (#cccccc, #ffffff)
  - 次要文本 (#808080)
  - 注释文本 (#6a9955)

### 排版系统
- **标题字体**: JetBrains Mono, Fira Code - 等宽字体，符合编程美学
- **正文字体**: Inter, -apple-system, BlinkMacSystemFont - 现代无衬线字体
- **代码字体**: Consolas, 'Courier New', monospace - 经典代码字体
- **字体大小层级**: 
  - H1: 2.5rem (40px)
  - H2: 2rem (32px)
  - H3: 1.5rem (24px)
  - 正文: 0.875rem (14px)
  - 小字: 0.75rem (12px)

## 视觉效果

### 使用的库和技术
- **Anime.js**: 实现流畅的UI动画和过渡效果
- **ECharts.js**: 创建数据可视化图表（如阅读统计、标签云等）
- **p5.js**: 生成动态背景和创意编程效果
- **Splide.js**: 实现文章轮播和图片画廊
- **Matter.js**: 添加物理动画效果
- **PIXI.js**: 高性能的2D渲染和视觉特效
- **Shader-park**: 创建自定义着色器背景效果

### 动画效果
- **目录树动画**: 平滑的展开/折叠动画，节点添加/删除动画
- **标签页动画**: 标签页创建、关闭、切换的流畅过渡
- **迷你地图**: 滚动同步的实时更新动画
- **页面加载**: 渐进式内容加载动画
- **悬停效果**: 微妙的3D倾斜、发光边缘效果
- **滚动动画**: 视差滚动和元素进入动画

### Hero区域效果
- **代码雨背景**: 类似黑客帝国的字符雨效果
- **粒子系统**: 动态的粒子连接网络
- **着色器背景**: 流动的几何图案和光线效果
- **终端模拟**: 模拟终端命令行的动态效果

### 样式特色
- **卡片设计**: 使用微妙的阴影和边框创建层次感
- **代码高亮**: 语法高亮风格的文本样式
- **图标系统**: 使用代码相关的图标（文件夹、文件、终端等）
- **状态指示**: 通过颜色和小图标表示不同的内容状态
- **响应式布局**: 适配不同屏幕尺寸的流体布局

### 交互反馈
- **微交互**: 按钮点击、悬停的视觉反馈
- **加载状态**: 优雅的加载动画和进度指示
- **错误处理**: 友好的错误提示和状态恢复
- **键盘导航**: 支持快捷键操作的视觉指示
